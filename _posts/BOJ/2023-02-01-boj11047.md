## 🔍 문제
> <https://www.acmicpc.net/problem/11047>
<div class="notice" markdown="1">
준규가 가지고 있는 동전은 총 N종류이고, 각각의 동전을 매우 많이 가지고 있다.

동전을 적절히 사용해서 그 가치의 합을 K로 만들려고 한다. 이때 필요한 동전 개수의 최솟값을 구하는 프로그램을 작성하시오.
</div>

## 🎯 풀이
```cpp
#include <iostream>

using namespace std;
int arr[10] = {0,};

int main(){
    int N, K;
    cin >> N >> K;

    for(int i=0; i<N; i++){
        cin >> arr[i];
    }
    int cnt = 0;

    while(K!=0){
        for(int i=N-1; i>=0; i--){
            if(K==0) break;
            int rem = K/arr[i];
            K %= arr[i];
            cnt += rem;
        }
    }
    cout << cnt;
}
```
