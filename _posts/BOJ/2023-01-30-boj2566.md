## 🔍 문제
> <https://www.acmicpc.net/problem/2566>
<div class="notice" markdown="1">
<그림 1>과 같이 9×9 격자판에 쓰여진 81개의 자연수 또는 0이 주어질 때, 이들 중 최댓값을 찾고 그 최댓값이 몇 행 몇 열에 위치한 수인지 구하는 프로그램을 작성하시오.

예를 들어, 다음과 같이 81개의 수가 주어지면

-생략-

이들 중 최댓값은 90이고, 이 값은 5행 7열에 위치한다.
</div>

## 🎯 풀이
```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int main() {
	int arr[9][9];

	for (int i = 0; i < 9; i++) {
		for (int j = 0; j < 9; j++) {
			cin >> arr[i][j];
		}
	}
	
	int max = 0;
	int row = 0;
	int column = 0;

	for (int i = 0; i < 9; i++) {
		for (int j = 0; j < 9; j++) {
			if (max < arr[i][j]) {
				max = arr[i][j];
				row = i;
				column = j;
			}
		}
	}

	cout << max << "\n";
	cout << row+1<< " " << column+1;
	return 0;
}
```
