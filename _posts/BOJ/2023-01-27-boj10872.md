## 🔍 문제
> <https://www.acmicpc.net/problem/10872>
<div class="notice" markdown="1">
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.
</div>

## 🎯 풀이
### 1. 첫 번째 풀이 ⭕
```cpp
#include <iostream>
int factorial(int num){
	if(num==0) return 1;
	if(num>2){
		num *= factorial(num-1);
	}
	return num;
}

int main()
{
	using namespace std;
	int N;
	cin >> N;
	cout << factorial(N);
}
```
