## 🔍 문제
> <https://www.acmicpc.net/problem/15651>
<div class="notice" markdown="1">
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.

- 1부터 N까지 자연수 중에서 M개를 고른 수열
- 같은 수를 여러 번 골라도 된다.
</div>

## 🎯 풀이
### 1. 첫 번째 풀이
```cpp
#include <iostream>

using namespace std;
int n, m;
int arr[10];

void func(int cnt){
  if(cnt == m){
    for(int i=0; i<m; i++){
      cout << arr[i] << ' ';
    }
    cout << '\n';
    return; // else가 없으므로 꼭 붙여줘야 함
  }
  for(int i=1; i<=n; i++){
    arr[cnt] = i;
    func(cnt + 1);
  }
}

int main(){
  cin >> n >> m;
  func(0);
}
```

## ✏️ 회고
백트래킹 알고리즘을 이해하기 위해 N과 M 문제를 풀기 시작했다.
